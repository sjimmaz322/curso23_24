Crea un programa con una función recursiva que muestre todos los números/letras capicúas encontrados en un array. Se entiende por capicúa cuando tiene al menos 3 posiciones del array. Por ejemplo con el array: 5 4 6 5 6 4 3 O S O

mostrará:
656
46564
OSO

Pista: const animals = ['ant', 'bison', 'camel', 'duck', 'elephant']; console.log(animals.slice(2)); // expected output: Array ["camel", "duck", "elephant"] console.log(animals.slice(2, 4)); // expected output: Array ["camel", "duck"] console.log(animals.slice(2, 4).reverse()); // expected output: Array ["duck","camel"]
